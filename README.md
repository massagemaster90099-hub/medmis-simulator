# 🏥 MedMIS Simulator

Тренажёр МИС для студентов-медиков. Безопасная среда для отработки навыков работы с медицинскими информационными системами.

## 🔒 Не нарушает 152-ФЗ 

**Все данные в проекте — синтетические.**
- Никаких реальных данных пациентов
- Все имена, диагнозы, даты — сгенерированы случайно
- Маркер `"is_synthetic": true` в каждой записи

## 🛠 Стек

- HTML/CSS/JS (фронтенд)
- GitHub Pages (хостинг)
